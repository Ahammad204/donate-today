## About

**Donate Today** is a lightweight, static charity website designed to inspire and facilitate donations for humanitarian causes. The site features a clean, modern UI with an auto-sliding hero banner, interactive donation amount selection, and a compelling narrative about giving back to those in need.

Built entirely with vanilla HTML, CSS, and JavaScript — no frameworks, no build tools, no dependencies.

## Features

- **Auto-Sliding Hero Banner** — Two rotating slides with smooth transitions (10s interval)
- **Interactive Donation Section** — Preset amounts (Tk 100 – 5000) with custom input support
- **Donation Plans** — Three impactful categories: Food, Shelter, and Education
- **Storytelling Blog** — Four alternating-layout blog entries highlighting real causes
- **Newsletter Subscription** — Email signup form for updates
- **Fully Responsive** — Optimized for mobile, tablet, and desktop screens
- **Zero Dependencies** — Pure vanilla stack, no frameworks or libraries required

## Getting Started

This project requires **no build step**. To run locally:

```bash
# Clone the repository
git clone https://github.com/Ahammad204/donate-today.git

# Open directly in browser
open index.html

# Or use a local dev server (recommended)
npx serve .
```
